[なぜホームページを編集する？]  
・年度決算計算書類をHOPS等経理委託先にて作成後，決算期の翌日から3ヶ月以内（6月末まで）に一般社団法人アケルの定時社員総会にて計算書類を承認する。  
・承認後速やかに貸借対照表の全文を法人のホームページ（HP）にて公告（電子公告）する。  
・一般社団法人及び一般財団法人に関する法律第128条第1項にて規定される法定義務。  
・なお，公告方法は登記情報であり，世界的大規模なネットワーク障害やHPドメインが閉鎖しHP再建に時間を要するなど，やむを得ない事情が無い限り，法人独自の判断で他の公告方法を取ってはならない。  
➡一般社団法人アケルのHPは無料ドメインであるGithub pagesにて作成しており，HPの編集のためにHTMLを一部編集する必要があるが，可能な限り簡易に編集できるように作成したため以下の「操作マニュアル」に従い操作されたし。
